# AnalisisAlgoritmos

Implemente el algoritmo de Kruskal y Dijkstra utilizando un lenguaje de programación de su elección. El objetivo es obtener el árbol de recubrimiento mínimo de un grafo no dirigido, y dirigido, conexo y ponderado, identificando las aristas que forman parte de la solución y calculando el costo total del árbol generado.

Deberá trabajar con un grafo propio, que debe estar en un archivo plano. La solución debe incluir el código fuente, un informe breve con la explicación del algoritmo, el grafo utilizado, la ejecución paso a paso, capturas de funcionamiento y una reflexión personal sobre la implementación desarrollada.
